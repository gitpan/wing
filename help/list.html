Help for message list screen

The message screen looks something like this:

Mailbox `foo' with 3 messages

 O   1. Sep 21 Fred Bloggs Re: Feeding the cat S R F D
D    2. Sep 28 Sam Smith Meeting tomorrow S R F U
 O   3. Sep 28 Nicky Jones Arrangements for Saturday S R F D
  • Clicking on a message subject displays it
  • Letters to the left of each line indicate when the message is tagged as (N)ew, (O)ld, (D)eleted or (F)lagged. Messages tagged as (D)eleted are not actually purged until the Purge button is pressed. Until that time, they can be untagged by using the (U)ndelete short cut (q.v.) or by displaying the message and using the Undelete button on that screen. Messages which are (F)lagged are those which are found to match a search, via the "Search" screen or the "Quick Search" field (q.v.), or have been flagged by the bulk operations drop down list (q.v.).
  • The letters to the right of each message are short cuts for (S)ave, (R)eply, (F)orward and (D)elete/(U)ndelete. Those functions are also available by displaying the message and using the correspondingly named buttons on that screen.
  • The upward and downward pointing arrows (or the words Prev and Next) take you to, respectively, the previous and next screenful of messages. You can configure how many message count as one screenful by using the Options button. The Top and Bottom buttons take you directly to the top or bottom of the mailbox. If you are already at the top or bottom of the mailbox, the unnecesssary buttons are greyed out or not displayed at all.
  • The Compose button takes you to the message composition screen where you can compose and send new messages.
  • The Mailboxes button takes you to a screen where you can select a different mailbox to view or create, rename or delete mailboxes and directories.
  • The Manage button takes you to a screen where you can manage your account: display your disk quota and disk usage, change your password, set mail forwarding or change and activate/deactiveate a vacation message.
  • The Options button lets you configure various options for either the current session or for every future session too.
  • The Purge button deletes from your mailbox all messages that are tagged as (D)eleted. Once purged, messages cannot be recovered. Once messages have been tagged as deleted, pressing the Purge button deletes them all immediately-there is no further confirmation dialog.
  • The Address Books button takes you to a screen where you can create, browse, search, update and share your address books.
  • The Logout button brings up a confirmation screen which lets you then logout of your mail session. You must remember to logout when you have finished your session.
  • The Search button brings up a screen on which you can search for and flag those messages in the current mailbox which contain your choice of words in a given header or the entire message text.
  • The Calendar button brings up a screen with a calendar of the previous/current/next months displayed along with today's date. You can also choose other months to display.
  • The drop down listbox (which initially contains the command "Search mailbox") and the input field to the right of it allow you to do a "Quick Search" of the current mailbox or to do various bulk operations.

    With the command "Search mailbox" displayed (as it is by default), you can type a word into the input field and press the "Go" button to search for messages containing that word. Messages which contain the entered word anywhere in their entire message text (headers or body) will be "Flagged" (marked with an "F" as described above). Pressing the "Go" button without entering a word will unflag all messages.

    With the drop down box set to "Delete range", "Undelete range", "Save range" or "Copy range" you can enter a numeric range of messages and, on pressing "Go", the chosen operation will be carried out on that range of messages. A range should be a comma-separated list of "a-b" entries or else the word flagged to refer to all those messages which are flagged. For example:

    10-20
    Act on messages 10 to 20 (inclusive).
    10-20,30-end
    Act on messages 10 to 20 (inclusive) and 30 and up (i.e. all messages from number 30 onwards). You can also use "last" or "*" instead of "end" to mean the same thing.
    flagged
    Act on all messages which are flagged (i.e. have an "F" next to them). You can abbreviate this word to the first letters of flagged or found to mean the same thing (e.g. "fla", "fou" or simply "f").
  • With the drop down box set to "Flag range" you can flag all messages in a given numeric range. The special range specifier flagged mentioned above would be meaningless in this situation and is not allowed.
