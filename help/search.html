Help for search screen

The search screen looks something like this:

Search mailbox

Search current mailbox for messages which match all the following conditions:
Subject
Message text
From
To

This screen lets you search through the current mailbox by looking for words that appear in message headers (Subject, From or To) or the entire Message text (consisting of all the headers and the body). Messages which match the text that you type are Flagged --they appear in the message list screen with an "F" beside them. All searching is done in a case insensitive way and a match is found if the exact text you type appears in the appropriate place in a message: there are no wildcards, pluses, minuses, fuzzy matching or anything complicated like that. Just type the text you want to find.

There is a drop down list from which you can select three variations of a basic search:

Only mark matching messages
This unflags all messages in the mailbox before starting the search.
When the search is over, the only messages flagged are those which matched your search conditions. This is sometimes called a "new" search.
Restrict search to currently marked messages
With this variation chosen, only those messages already flagged will be looked at at all. This lets you restrict a previous search even further by only considering for matching those messages matching your previous condition. This is sometimes called an "and" search.
Mark matching messages in addition to those currently marked
With this option, messages matching your search conditions will be flagged, leaving any previously flagged messages with their flag intact. This lets you add more messages to a previous search. This is sometimes called an "or" search.

Once you press the Search button and the search is complete, you are returned to the message list screen with found messsages flagged with an "F". You can restrict or add messages by returning to the "Search" screen and choosing a "Restrict..." or "Mark ... in addition" search type.

On the message list screen you can also do bulk operations (delete, save, copy) on those flagged messages by using the drop down list box on the top right of that screen.
